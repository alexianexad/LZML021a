Analyse des données dans un fichier

Voici un mode d'emploi :

  • Etape 1 : Chargez un texte au format .txt à l'aide du bouton Parcourir.
  • Etape 2 (facultatif) : Précisez les délimiteurs de votre choix puis cliquez sur le bouton Segmentation.
  • Le bouton Dictionnaire : permet d'obtenir un tableau qui classe les tokens par fréquence décroissante.
  • Le bouton GREP : entrez une regex ou un mot et cliquez pour voir les lignes où il apparaît.
  • Le bouton Concordancier : comme GREP mais avec contexte gauche/droite.
  • Le bouton Nombre de phrases : affiche combien de phrases contient le texte.
  • Le bouton Mots les plus longs : montre les 10 mots les plus longs.
  • Le bouton Pie Chart : génère un camembert des mots les plus fréquents (hors stopwords).
  • Le bouton /kujuj/ : ajoute "uj" à la fin de chaque mot du texte.
